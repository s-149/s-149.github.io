      1. Target folder, "%temp%, C:\Windows\Temp, C:\Windows\Prefetch, Recycle Bin"
      2. folder diatas adalah tempat penyimpanan sementara, cache, dan sampah
      3. pastikan file yg ada di dalam folder "Recycle Bin" tidak akan digunakan kembali karena akan didelete permanent.
